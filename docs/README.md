# PANEPISTIMIAKOS :: Πλατφόρμα διαχείρησης φοιτητικών events 

Το συγκεκριμένο project προορίζεται να παρουιαστεί ως εργασια στο Πανεπιστήμιο Ιωαννίνων στο τμήμα Πληροφορικής & Τηλεπικοινωνιών. Ειναι μια πλατφορμα διαχειρησης φοιτιτικων εκδηλωσεων που δινει τη δυνατοτητα στον κάθε χρήστη να: δημιουργήσει, τροποποιήσει, διαγράψει και να εγγραφεί σε εκδηλώσεις που δημιουργούν άλλοι φοιτητές. 

Υπάρχει η δυνατότητα αξιολόγησης των εκδηλώσεων και η αποστολή σχολίων ως feedback. Η εγγραφη σε μια εκδήλωσει μπορει να πραγματοποιηθει εφοσον ισχύουν οι προυποθέσεις: Ο χρήστης έχει λογαριασμό, η εκδήλωση δεν έχει λήξει (με βάση την ημερομηνία).

Το backend της πλατφορμας διαχειριζεται απο το Subabase στο οποιο υπάρχουν όλα τα σχετικά στοιχεία των χρηστών καθως και οι πινακες στους οποιους αποθηκεύονται τα στοιχεια κρυπτογραφημένα.
